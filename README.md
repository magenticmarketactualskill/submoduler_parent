# SubmodulerParent

Parent gem for Submoduler - Git submodule management tool.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'submoduler_parent'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install submoduler_parent
